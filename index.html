Introdução ao HTML na prática

voltar

HTML (Hypertext Markup Language) é a linguagem padrão para criação e estruturação de páginas web. Neste tutorial, vamos explorar a estrutura básica do HTML e alguns conceitos fundamentais. 

            
                <html>
                    <head>
                        <title>Meu primeiro HTML</title>
                    </head>
                    <body>
                        Meu primeiro HTML!
                    </body>
                </html>

A estrutura básica do HTML consiste nas tags hmtl, head e body. A tag head contém informações pré-carregadas, enquanto a tag body contém o conteúdo visível ao usuário.

Introdução a Tags

voltar

As tags são elementos essenciais no HTML, permitindo a marcação e formatação do conteúdo. 

            
                <html>
                    <head>
                        <title>Meu primeiro HTML</title>
                    </head>
                    <body>
                        <strong>Meu primeiro <i>HTML</i>!</strong>
                        <input type="text" />
                    </body>
                </html>

Além das tags de formatação, existem tags que não requerem uma tag de fechamento. Algumas, como input type="text" e img, são auto-fecháveis.

Atributos das Tags

voltar

Atributos fornecem propriedades adicionais às tags, sendo globais ou específicos de cada tag. 

            
                <strong id="titulo">Meu primeiro HTML!</strong>
                <strong style="color: blue;">Meu primeiro HTML!</strong>
                <strong class="titulo-principal">Meu primeiro HTML!</strong>

A tag strong exemplifica atributos como "id" para identificação no JavaScript, "style" para estilos inline e "class" para aplicar estilos CSS padronizados. 

            
                <input type="text" />
                <input type="number" />
                <input type="color" />

A tag input possui atributos específicos, como "type," determinando o tipo de entrada. 

            
                <img width="250" src="https://observatorio3setor.org.br/wp-content/uploads/2019/09/1-girassol.jpg"/>

A tag também utiliza atributos como "width" e "src" para definir largura e fonte da imagem.
